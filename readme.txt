Add 'things' to your favorites

== Description ==

Gives the website user an option to add things to their favorites. ( And then remove it, if required ! )

**Works with :**

*   Posts
*   Pages
*   Custom Post Types
*   Taxonomies
*   Members�


So the website user can add posts, pages, CPT\'s, taxonomies and even other members to their favorites list.

**Possible usage:**

*   As a 'favorite posts', 'favorite articles', 'favorite products', etc. plugin
*   As a mechanism to maintain a list of 'most liked' content on your website.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

Then go to the plugin settings page inside 'Settings' menu and configure some option to make the plugin work best according to your requirement.
